# 外星人入侵 (Alien Invasion)

基于Pygame开发的经典2D射击游戏，实现飞船控制、外星人自动射击、升级子弹奖励、动态难度调整等功能，是Python游戏开发的入门实践项目。

## 项目特点
- 核心玩法：玩家操控飞船射击外星人舰队，躲避敌方子弹，随关卡提升难度
- 扩展功能：得分累计奖励散射子弹、实时计分系统、多场景音效反馈
- 技术亮点：面向对象封装游戏元素、Sprite精灵组优化碰撞检测、浮点数坐标实现平滑移动

## 运行环境
- Python 3.8+
- Pygame 2.0+
